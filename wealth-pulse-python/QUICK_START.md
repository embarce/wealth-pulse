# 股票信息初始化 - 快速指南

## 📋 概述

使用手动初始化脚本从 Excel 文件批量导入股票数据到数据库。

### 数据供应商配置

系统支持两种数据供应商，通过 `.env` 文件配置：

```bash
# 方案 1: Yahoo Finance（默认，支持全球市场）
STOCK_DATA_PROVIDER=yfinance

# 方案 2: AkShare（推荐用于国内股票，访问更快）
STOCK_DATA_PROVIDER=akshare
```

**选择建议**：
- 仅投资 A股/港股 → 使用 `akshare`（更快）
- 全球投资组合 → 使用 `yfinance`（更全面）

详细说明请参考：[STOCK_DATA_PROVIDER_USAGE.md](./STOCK_DATA_PROVIDER_USAGE.md)

---

## 🚀 快速开始（3 步）

### 步骤 1：准备 Excel 文件

**选项 A：生成示例文件**
```bash
python create_sample_excel.py
```

**选项 B：手动创建 `hk-code.xlsx`**

| Code     | Name       |
|----------|-----------|
| 0700.HK  | 腾讯控股   |
| 9988.HK  | 阿里巴巴   |
| NVDA.US  | 英伟达     |

⚠️ **格式要求**：
- 表头必须是 `Code` 和 `Name`（英文）
- Code 格式：港股 `.HK`，美股 `.US`

### 步骤 2：试运行（推荐）

```bash
python init_stocks.py --dry-run
```

预览将要执行的操作，不实际写入数据库。

### 步骤 3：正式执行

```bash
python init_stocks.py
```

输入 `yes` 确认后开始写入数据库。

## 🔄 自动更新

初始化完成后，定时任务会自动更新数据：

- ✅ **每 5 分钟**：更新价格、涨跌幅
- ✅ **每天 8:00 AM**：更新公司信息

## ⚙️ 命令行参数

| 参数 | 说明 |
|------|------|

